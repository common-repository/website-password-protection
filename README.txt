== Description ==

You can restrict customer access to your online store by enabling the password page. The password page is a landing page that adds password protection to your online store. You can create a password and give it to only the customers that you want to be able to visit your store.

**Key Features**

* Add password protection to your online store.
* Remove password protection from your online store.
* Change your password page settings
* Hide the footer on your password page
* Hide your store name on your password page
* Edit the code templates for your password page


== Installation ==

**Installation Overview**

**1. Install and activate the plugin**
In your WordPress dashboard, go to Plugins => Add New. Search for "Website Password Protect", and click Install Now.
After the installation process completes, click the Activate Plugin link to activate the plugin.

**2. Setup Password**
Go to Settings => Reading. Scroll all the way down to "Password Protection". Enter your password and click Save Changes button.

**3. Remove Password**
**2. Setup Password**
Go to Settings => Reading. Scroll all the way down to "Password Protection". Remove password and click Save Changes button.

Go to Settings => Reading. Scroll all the way down to "Password Protection". Enter your password and click Save Changes button.
